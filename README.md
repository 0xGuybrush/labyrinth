# labyrinth

An implementation of [DigitasLBi's Labyrinth Challenge](http://labyrinth.digitaslbi.com) in Javascript.

## Installation ##

```
npm install
gulp
node dist/index.js
```

## Dependencies ##
 - Node >= 0.12 
 - Babel (Transpiling of ECMAScript 6 features)
 - Gulp (Task runner for build process)
